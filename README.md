# SPLLIT React

A modern React application built with Vite, TypeScript, and Tailwind CSS.

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
